Kaikki sceanes kuten Menu ja Peli löytyy Assets/Scenes.
Joilakin ojekteilla sceanissä voi vaihtaa numeroita jos haluaa.

Kaikki scriptit löytyy Assets/Assets/Scripts. 
Kaikki variables jolla on Public sana voi vaihtaa editorissa jos valitset sen objektin.

Kaikki materials löytyy Assets/Assets/Materials. Voit vaihtaa niiden väriä ja muita asetuksia.

Voit löytää kameraefektin Assets/Assets/CameraEffect. Voit vaihtaa kaikenlaisia asetuskia kameraefektiin Custom_Img tiedostossa.

Voit vaihtaa keybindings Edit>Project Settings>Input Manager. Enintään Vertical ja Vertical2 säätää paddle liikettä.

Voit rakentaa projektin EXE tiedostoksi File>Build Settings ja Build. Voit vaihtaa kaikenlaisia asetuksia Edit>Project Settings>Player>Resolution and Presentation.
Melkein aina haluat käyttää Exclusive Fullscreen ja Default is Native Resolution.
